# PROMPTS

## 开发前提示词

请先扫描当前项目状态，再开始修改。

要求：
- 只做一个最小闭环
- 先列出待修改文件
- 只修改与当前任务直接相关的文件
- 不做无关重构
- 修改完成后更新 `ai-memory/DEV_MEMORY.md`
- 如果产生新的技术结论或约束，同步更新 `ai-memory/DECISIONS.md`
- 输出保持简洁

## 开发后更新提示词

本次开发完成后，请同步更新 AI 记忆文件。

至少更新：
- `ai-memory/DEV_MEMORY.md`
- `ai-memory/TASKS.md`
- `ai-memory/DECISIONS.md`

更新内容包括：
- 当前开发阶段
- 已完成事项
- 未完成事项
- 最近修改文件
- 风险点
- 下一步建议
- 本次新增技术决策

## 扫描项目提示词

请扫描当前项目并输出以下内容：

1. 当前开发阶段
2. 已完成功能
3. 未完成功能
4. 当前模块涉及的 controller / service / mapper / xml / SQL / frontend 文件
5. 数据库结构与代码映射是否一致
6. 存在的风险点
7. 下一步最合理的开发顺序

扫描要求：
- 优先结合实际文件
- 说明具体文件路径
- 不要泛泛而谈
- 如果数据库实例状态未知，要明确写“未知”
