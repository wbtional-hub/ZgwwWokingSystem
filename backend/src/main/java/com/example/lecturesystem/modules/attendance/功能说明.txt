模块名称：attendance

功能描述：
考勤模块。负责上班/下班打卡、定位校验、有效性判定、考勤记录查询与导出。与 param、permission、statistics 模块有关系。

与其他模块关系：
1. 本模块为独立文件夹，不与其他模块混写。
2. 若涉及用户身份、角色、组织范围，则调用 auth / permission / orgtree。
3. 若涉及审计留痕，则调用 operationlog。
4. 需要参数控制时，调用 param。
5. 统计模块通常消费 attendance / weeklywork / orgtree 的结果。

开发约束：
1. Controller 只做收参与返回。
2. Service 实现业务。
3. DTO 只放请求参数。
4. Entity 只放数据库字段。
5. 后续新增 mapper、xml、vo 时，也放在当前模块文件夹内。
